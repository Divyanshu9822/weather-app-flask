# Weather App with Flask

This is a simple Weather App built using Flask and the OpenWeather API. It allows users to check the weather for a specific city or view the weather for Paris by default when landing on the page.

## Features

- **Search by City:** Users can search for weather information in a specific city.
- **Default View:** When a user lands on the page, they are presented with weather data for Paris.
- **Error Handling:** The app provides an error message if the user enters an incorrect city name.
- **Current Date Display:** The current date is displayed in the format `dayname date month year`
- **Weather Icon:** Weather conditions are illustrated with icons.
